## 2D Line Clipping
#### Cohen sutherland algorithm
1. Mouse interfacing is used for line drawing and window drawing
1. Cohen sutherland algorithm is used to clip each line based on three cases
	1. Fully visible
	1. Fully invisible
	1. Partially visible
